# Use cases

## Personal productivity
- daily plan, status updates, meeting notes, follow-ups

## Developer assistant
- draft PR descriptions, release notes, changelogs
- generate test plans, checklists, QA scenarios

## Ops helper
- summarize logs pasted in chat
- runbook drafting (manual unless you connect tools)

## Edge/IoT (PicoClaw)
- home lab assistant (alerts, summaries, routines)
