# Personal Agent Starter Kit

Build your own personal AI agent using OpenClaw, Nanobot, or PicoClaw.

## Choose your stack in 60 seconds

- Full assistant platform → OpenClaw
- Lightweight Python assistant → Nanobot
- Edge / IoT assistant → PicoClaw

Quick start:

```bash
make doctor
make setup_nanobot
```

## Dev environment (Docker)

```bash
docker compose run --rm dev bash
make doctor
```
